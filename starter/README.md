# Starter Files

This folder contains the scaffolded project files for the StyleSense recommendation prediction pipeline.

## Contents

- `starter.ipynb` - Main Jupyter notebook with the ML pipeline (EDA, preprocessing, training, tuning)
- `data/reviews.csv` - Dataset with 18,442 customer reviews and product information
- `models/recommendation_pipeline.pkl` - Saved trained model (pickle file)

## Getting Started

1. Activate the virtual environment: `.\p3_ds\Scripts\Activate.ps1`
2. Install spaCy model: `python -m spacy download en_core_web_sm`
3. Open the notebook: `jupyter notebook starter.ipynb`
